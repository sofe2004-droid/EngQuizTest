## 🎯 상황별 추천

**→ PythonAnywhere**

### 2️⃣ Google AI API를 무료로 사용하고 싶은 경우
**→ Render**
- 완전 무료

**→ PythonAnywhere**
- Google API 기능만 제외하고 테스트
- 나중에 Render로 이전 가능

### 4️⃣ 실제 서비스로 운영하고 싶은 경우
**→ Render 또는 Railway**
- 안정적인 무료 플랜
- 자동 재배포
- 프로덕션 환경 지원

## 🔍 상세 비교

### PythonAnywhere

#### ✅ 장점
- **가장 쉬움**: 웹 인터페이스만으로 배포
- **Git 불필요**: 파일 업로드만 하면 됨
- **Python 특화**: Flask에 최적화
- **무료 플랫**: 기본 기능 사용 가능

#### ❌ 단점
- **외부 API 제한**: 무료 플랜은 화이트리스트만 접속 가능
- **Google AI 사용 불가** (무료 플랜)
- **CPU 시간 제한**: 하루 100초
- **수동 업데이트**: 파일 재업로드 필요

#### 💰 비용
- 무료: 제한적 (Google API 불가)
- $5/월: 외부 API 제한 해제

#### 📝 배포 가이드
→ [DEPLOY_PYTHONANYWHERE.md](DEPLOY_PYTHONANYWHERE.md)

---

### Render

#### ✅ 장점
- **완전 무료**: Google API 사용 가능
- **자동 재배포**: Git push만 하면 자동 배포
- **제한 없음**: 외부 API 자유롭게 호출
- **HTTPS 자동**: 보안 인증서 무료
- **Sleep 모드**: 비용 절약

#### ❌ 단점
- **Git 필요**: GitHub 계정 및 Git 사용법 필요
- **초기 설정**: YAML 파일 및 설정 필요
- **Sleep 모드**: 15분 미사용 시 sleep (재시작 30-50초)

#### 💰 비용
- 무료: 완전 무료 (제한 관대)
- $7/월: Sleep 모드 없음

#### 📝 배포 가이드
→ [DEPLOY.md](DEPLOY.md)

---

### Railway

#### ✅ 장점
- **간단함**: Render보다 설정이 간단
- **$5 크레딧**: 무료로 제공
- **빠른 배포**: 2-3분 안에 완료
- **자동 재배포**: Git push로 자동

#### ❌ 단점
- **크레딧 제한**: $5 소진 후 유료
- **Git 필요**: GitHub 계정 필요

#### 💰 비용
- 무료: $5 크레딧 (약 1-2개월 사용 가능)
- $5/월: 크레딧 소진 후

---

### Vercel (고급)

#### ✅ 장점
- **빠른 속도**: 전 세계 CDN
- **자동 HTTPS**: 무료 SSL
- **자동 재배포**: Git 연동

#### ❌ 단점
- **Serverless 필요**: Flask를 Serverless로 변환 필요
- **복잡한 설정**: 초보자에게 어려움
- **SQLite 불가**: 데이터베이스 외부 사용 필요

---

### Netlify ❌

#### 왜 불가능한가?
- **정적 사이트만**: HTML/CSS/JS만 호스팅
- **Python 미지원**: Flask 서버 실행 불가
- **데이터베이스 불가**: SQLite 사용 불가
- **파일 쓰기 불가**: CSV, JSON 저장 불가

#### 대안
- 프론트엔드만 Netlify
- 백엔드는 Render/Railway
- ⚠️ 하지만 전체 리팩토링 필요
